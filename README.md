# Tic-Tac-Toe Game

This repository contains a simple command-line Tic-Tac-Toe game written in Python.
You can play against a basic AI that selects random moves.

## Running the Game

Use Python 3.8 or later. To start the game, run:

```bash
python3 -m src.tictactoe
```

## Running Tests

The project uses the built-in `unittest` module. Discover and run all tests with:

```bash
python3 -m unittest discover
```

## Web Version

Open `web/index.html` in a browser to play a web version of the game styled with a cyberpunk theme.
